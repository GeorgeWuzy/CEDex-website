CEDex Dataset

The CEDex dataset consists of two object sets:

  • Real-World Object Set: Contains objects from the ContactDB and YCB datasets with 200K grasps.
  • Synthesis Object Set: Includes over 500K synthesized objects, generating 20M grasps in total.
